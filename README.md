# Spec360 Communication – Full‑Stack Website

[![Live Demo](https://img.shields.io/badge/demo-live-brightgreen)](https://spec360-client.vercel.app)
[![Backend Status](https://img.shields.io/badge/backend-render-blue)](https://spec360-server.onrender.com)

A modern, responsive business website for **Spec360 Communication** – a tech and digital solutions brand. Built with React (Vite), Tailwind CSS, Framer Motion, Node.js, Express, and MongoDB.

## Features

- **Landing Page** with animated sections (Hero, About, Services, Vision & Mission, Core Values, Brand Promise, Contact).
- **Service Pages** for each offering (Phones & Accessories, Repairs, Web Development, POS, Connectivity).
- **E‑commerce Store** for phones and accessories (add to cart, cart page, product details).
- **Admin Dashboard** with authentication (JWT) – manage products (add, delete).
- **Contact Form** with email integration (Nodemailer).
- **Image Uploads** to Cloudinary.
- **Fully Responsive** design, dark theme with red accents.

## Tech Stack

**Frontend**
- React (Vite)
- Tailwind CSS
- Framer Motion (animations)
- Lucide React (icons)
- React Router DOM
- Axios

**Backend**
- Node.js, Express
- MongoDB with Mongoose
- JWT Authentication
- Bcrypt.js
- Cloudinary, Multer
- Nodemailer
